# Hand-tracking-project
Computer vision project for real-time hand landmark detection and tracking.
This project demonstrates real-time hand tracking using a webcam, OpenCV, and MediaPipe. The application detects hand landmarks and visualizes hand movements live, providing a foundation for gesture recognition, virtual controls, and other computer vision applications.
built using Python, OpenCV, and MediaPipe.

## Features

- Detects hand landmarks using a webcam
- Tracks hand movement in real time
- Displays hand connections and key points

## Technologies Used

- Python
- OpenCV
- MediaPipe

## How It Works

The application captures video from a webcam using OpenCV. MediaPipe processes each frame and detects hand landmarks. The landmarks are then displayed in real time on the video feed.
